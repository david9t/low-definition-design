Sections and blocks

This is a section

It generally stretches the full width of the browser.

It has applied to it the "section--separated" class, which creates a dividing border, surrounding the section.

Background colours, patterns or images can be used to distinguish between sections with finished designs.

This is a block

A block should generally be used for smaller gatherings of breakout content.

Multiple blocks can exist within a single section.

In a finished design, a block can use a border, shading or a unique background colour to allow it to stand out from the section it sits within.

Naturally blocks will stretch to the full width of the browser, like this one.

Expansion of section and block spacing

Going beyond the natural, minimal best practice spacing built into LO-DEF, we can choose to let our sections and blocks get larger when screen size allows it:

Natural section spacing

This section doesn't have an optional spacing class applied to it, so falls back to the minimal best practice spacing.

Natural block spacing

This block doesn't have an optional spacing class applied to it, so falls back to minimal best practice spacing.

Luxury block spacing

This block uses luxury spacing.

Starting mobile-first it won't have any affect, minimal best practice spacing will be used. As the browser gets wider and wider, the class kicks in and gives this block more padding.

Useful classes

Apply this class to structural tags:

  • .block--luxury-spacing

Oversized section spacing

This section uses oversized spacing.

Starting mobile-first it won't have any affect, minimal best practice spacing will be used. As the browser gets wider and wider, this class kicks in and gives this section more padding.

Check these at different browser sizes to observe different behaviour.

Optional limits to section and block spacing

We can also choose to limit the spacing of our sections and blocks.

These limits will apply consistently, regardless of screen size.

Base section spacing

This section is limited to base spacing.

Natural, global changes to section spacing won't take affect on this section.

Base block spacing

This block is limited to base spacing.

Natural, global changes to block spacing won't take affect on this section.
