Basic, natural typography

Globally defined, good practice typographic rules.

If nothing else, these basic principles will ensure a good quality, typogaphy-first approach to your design.

Modular scale

LO-DEF uses modular-scaled proportions between body copy and each level of heading. Inspired by typography-first thinking, modular scale values are also used when it comes to layout and spacing.

Global good practice

Loosely inspired by the visual order of baseline grids, LO-DEF is based on a unified vertical rhythm between all types of text.

We also set automatic limits to line length, based on ideal reading and scanning habits.

Text lockups

Lockup

This block has a "lockup" class applied to it. This means all text, regardless of size, will be limited to the same line-length.

Here's an example heading. It's line-length won't exceed that of any other text within the same area

No lockup

To illustrate the difference, here's the same SECTION again, without the "lockup" class applied:

This block does not have the "lockup" class applied to it. This means all text will be limited only by its own natural maximum line-length.

Here's an example heading. It's line-length will now exceed that of the other text within the same area

This heading also will be bound by a different maximum horizontal width, based on its own font size

Spacing limits
